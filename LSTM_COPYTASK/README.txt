# LSTM - COPYTASK


Este projeto implementa uma rede neural LSTM para resolver uma tarefa de cópia simplificada de sequência binária.

## Estrutura

- `dataset.py`: Gera os dados de treinamento e validação.
- `lstm_model.py`: Define a arquitetura da rede LSTM.
- `train.py`: Script principal para treinamento do modelo.

## Objetivo

Treinar uma LSTM para memorizar e reproduzir uma sequência de bits com desempenho satisfatório em tarefas simples de memória de curto prazo.

## Execução

```bash
python train.py
